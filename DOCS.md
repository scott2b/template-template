# template-template

This repository is a GitHub template used to create hybrid GitHub+Copier template repositories.
Use the **template/** directory as the canonical source for your template logic.
The repository root contains the rendered output for GitHub's template feature.


### Creating a new template repository

Clone this repo using GitHub's "Use this template", then edit it normally.
Run `./sync_root.sh` whenever you update the template logic in `template/`.
